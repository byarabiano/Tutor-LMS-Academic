# Tutor LMS Academic Pro

Advanced academic structure system for Tutor LMS with multi-level categories and content isolation.

## Features

- Multi-level academic categories (up to 10 levels)
- Content isolation between categories
- Integration with popular plugins
- Multisite support
- Export/Import functionality
